Mulancha v2

Mulancha v2の概要

  ・マウス操作型プログラムランチャーです
  ・起動した時のマウス移動が少なくなる「パイ型」が特徴です

必要環境

  ・HSP3なんで、たぶんWindows 2000以上？

起動方法

  ・mlc2.exeを実行すると、タスクトレイに入ります
  ・この状態で、右クリック押しながら左クリックを押し、両者を離すと画面が起動します
      これを「ロッカージェスチャ」と呼びます。VivaldiやOtter Browserとは競合するよ
  ・タスクトレイのアイコンを左クリックすると終了させる事ができます

操作方法

  ・アイコンがある所をクリック
      それを起動します。
      デフォルトではIE、notepad、mspaintが入っています。
  ・アイコンがある所を右クリック
      編集します。
  ・何もない所をクリック
      新規項目を作成します。
  ・何もない所を右クリック
      タスクトレイに待避します。
  ・アクティブな状態でEscキー
      終了します。

編集方法

  ・まずファイルを開くダイアログが出る
      起動したいexeを選んでください。
  ・編集画面が出ます
      以下の項目を入力して登録
      名前：識別できる名前を入力。ソフト名とか
      場所：自動入力されるはずです。起動したい場所を変更するなら参照ボタン等で変更
      Icon：自動入力されるはずです。アイコンを変更するなら参照ボタンで変更
            画像指定は無理
      種別：種類を選択する
            実行形式    ：実行ファイルとか
            ファイル    ：実行ファイルじゃない奴全般
            サブメニュー：クリックすると下にもう１個ランチャーが生成される奴
            管理者権限  ：regedit等、管理者権限で実行させたい実行ファイル

制限事項

  ・一つのパイは最大16分割
      増やしたければ適当にソースいじってください。
  ・Unicode非対応
      HSP@Windowsの宿命。なんとかしたい人はCSharpにでも移植してください。
  ・UWPアプリ、ほぼ非対応
      アイテムに登録するだけならできます。例えば「start ms-settings:」と叩いてみてください。
      アイコンは無理くさい。

TIPS

  ・UWPアプリを登録したい
      面倒臭いですが方法があります。以下Windows 10の場合です。
      UWPアプリを登録したいサブメニューをテキストエディタ(Shift-JIS)で開いてください。
      「name=」任意
      「path=」スタートメニュー＞設定＞アプリ＞アプリと機能＞アプリ実行エイリアス
               ここに記載のある「*.exe」を入力
      「icon=」エクスプローラーを開き、「shell:AppsFolder」を叩く
               登録したいUWPアプリを右クリック＞ショートカットの作成
               デスクトップに作成されたショートカットのパスを入力
               それか諦めて何かのアイコンのパスを入力
      「type=」exe
      保存して閉じます。
